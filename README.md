# Smart-Intonation
music software for intonation training

## Making it work

Install the QT5 and Qwt development packages:

```
    apt-get install qtdeclarative5-dev-tools
    apt-get install libqwt-qt5-dev
    
```

To clone this git repository.

To build:

```
    cd qwt-example
    cmake .
    make
```

To run:

```
    ./qwt-example
```
